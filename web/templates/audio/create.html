⚙️ Parâmetros avançados

Configurações salvas

🔢 Passos de difusão (16-64). Valores maiores = melhor qualidade mas mais lento. Padrão: 32. Recomendado: 16 para rápido, 32 para qualidade.
🎯 Força do guia (0.0-5.0). Controla quanto o modelo segue as instruções. Padrão: 2.0. Valores altos = mais fidelidade.
📊 Deslocamento temporal (0.0-1.0). Ajusta o agendamento de ruído. Padrão: 0.1. Raramente precisa alterar.
🎲 Temperatura de posição (0.0-10.0). 0 = determinístico (sempre igual), maior = mais variação. Padrão: 5.0.
🎲 Temperatura de classe (0.0-10.0). 0 = determinístico. Padrão: 0.0. Controla variação de tokens.
⚖️ Fator de penalidade (0.0-10.0). Penaliza camadas específicas. Padrão: 5.0.
⏩ Velocidade da fala (0.5-2.0). 1.0 = normal, >1 = mais rápido, <1 = mais lento. Ex: 1.2 = 20% mais rápido.
⏱️ Duração fixa em segundos. Se definido, ignora o parâmetro speed. Deixe vazio para duração automática.
✂️ Duração de cada pedaço de áudio (segundos). Textos longos são divididos. Padrão: 15s.
📏 Limite para dividir em pedaços (segundos). Se áudio estimado > threshold, divide. Padrão: 30s.
🌍 Código do idioma no formato ISO (opcional). Ex: pt-BR (português Brasil), en-US (inglês EUA), es-ES (espanhol), fr-FR (francês), zh-CN (chinês), ja-JP (japonês). Auto-detecta se vazio.
